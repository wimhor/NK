# NK
Code for performing random, uphill, or downhill walks on NK\(C) landscapes.

This code combines various functions that were used to generate the results in the following publications:
- W. Hordijk and S. A. Kauffman. [Correlation analysis of coupled fitness landscapes](http://onlinelibrary.wiley.com/doi/10.1002/cplx.20092/abstract). _Complexity_ 10(6):41-49, 2005.
- W. Hordijk, S. A. Kauffman and P. F. Stadler. [Average fitness differences on NK landscapes](http://link.springer.com/article/10.1007/s12064-019-00296-0). _Theory in Biosciences_ 139(1):1-7, 2020.

This code comes without any warranty or documentation, and is partially based on earlier C code by [Terry Jones](https://github.com/terrycojones/nk-landscapes). Feel free to use it for your own purposes, but if you do so, please include a reference (with explicit link) to the [current repository](https://github.com/wimhor/NK).
